# CodeJudge DBMS System

This project is a relational database design and SQL implementation for the CodeJudge platform dataset provided in the Masai School DBMS assignment.

## Project Objectives

- Design normalized relational schema
- Identify entities and relationships
- Apply primary and foreign keys
- Implement normalization concepts
- Create SQL schema using SQLite
- Document ER relationships and assumptions

## Technologies Used

- SQLite
- SQL
- VS Code
- Git & GitHub

## Repository Structure

- data/raw → original dataset CSV files
- database → schema and database files
- docs → DBMS explanations and normalization notes
- queries → SQL query files for assignment parts
- scripts → helper scripts

## Core Tables

- students
- batches
- courses
- enrollments

## Concepts Covered

- Primary Keys
- Foreign Keys
- Candidate Keys
- Normalization
- ER Modeling
- Relational Design
